🐍 Agente Organizador de Archivos con Python y Claude

Construye tu primer agente de Inteligencia Artificial con Python que escanea una carpeta desordenada, utiliza Claude de Anthropic para clasificar los archivos por categoría y los organiza automáticamente en subcarpetas. Todo desde Visual Studio Code.

🐍 Python 3.11 🤖 Anthropic Claude 💻 VS Code ⚡ Agente IA

Nivel: Intermedio | Archivos Python: 2 | Duración estimada: 60-90 minutos

Conceptos Clave

📖 ¿Qué es un Agente de IA?

Un agente de IA es un programa que puede percibir su entorno (leer archivos), razonar sobre lo que encuentra (usando un modelo de lenguaje como Claude) y actuar en consecuencia (mover, renombrar, crear archivos). A diferencia de un script tradicional que sigue reglas fijas, un agente usa IA para tomar decisiones flexibles.

🔄 Script Tradicional vs Agente con IA

📜 Script Tradicional

  • Clasifica por extensión: .pdf → documentos/
  • Reglas fijas codificadas
  • No entiende el contenido
  • foto_informe.jpg → va a imágenes (incorrecto)
  • Cada nueva extensión requiere código nuevo

🤖 Agente con Claude

  • Clasifica por nombre + contexto
  • Decisiones inteligentes y flexibles
  • Entiende la intención del archivo
  • foto_informe.jpg → puede ir a documentos
  • Se adapta sin cambiar código

🤖 ¿Por qué Claude para esta tarea?

Claude de Anthropic es ideal porque:

  • Entiende lenguaje natural: puede interpretar nombres de archivos como "tarea_final_matematicas_v2.docx"
  • Responde en JSON: le pedimos formato estructurado y responde exactamente así
  • Es seguro: Anthropic se enfoca en IA segura y controlable
  • API simple: solo necesitas una librería Python (anthropic)

🏗️ Arquitectura del Agente

Capa Componente Tecnología Función
Percepción escanear_carpeta() os, pathlib Lee los archivos del directorio
Razonamiento clasificar_con_claude() anthropic (API) Claude decide la categoría de cada archivo
Acción mover_archivos() shutil, os Crea carpetas y mueve archivos

Paso 1 — Escanear la Carpeta

El primer paso de nuestro agente es percibir su entorno: leer qué archivos hay en la carpeta.

1

Crear el archivo organizador.py

Crea un nuevo archivo en VS Code llamado organizador.py. Empezamos con la función de escaneo:

"""
Agente Organizador de Archivos con IA (Claude).
Escanea una carpeta, clasifica archivos con Claude y los organiza automáticamente.
"""
import os
import json
import shutil
from pathlib import Path

import anthropic


def escanear_carpeta(ruta_carpeta: str) -> list[str]:
    """
    Escanea una carpeta y retorna la lista de archivos (no carpetas).
    
    Args:
        ruta_carpeta: Ruta a la carpeta que se desea escanear.
    
    Returns:
        Lista con los nombres de los archivos encontrados.
    """
    carpeta = Path(ruta_carpeta)
    
    if not carpeta.exists():
        print(f"❌ Error: La carpeta '{ruta_carpeta}' no existe.")
        return []
    
    if not carpeta.is_dir():
        print(f"❌ Error: '{ruta_carpeta}' no es una carpeta.")
        return []
    
    # Listar solo archivos (no subcarpetas)
    archivos = [f.name for f in carpeta.iterdir() if f.is_file()]
    
    if not archivos:
        print(f"⚠️ La carpeta '{ruta_carpeta}' está vacía.")
        return []
    
    print(f"📂 Se encontraron {len(archivos)} archivos en '{ruta_carpeta}':")
    for archivo in sorted(archivos):
        print(f"   📄 {archivo}")
    
    return archivos


# --- Prueba rápida ---
if __name__ == "__main__":
    archivos = escanear_carpeta("desordenada")
    print(f"\nTotal: {len(archivos)} archivos")
2

Probar el escaneo

✅ Resultado Esperado

Verás la lista de los 23 archivos que creamos en el paso anterior.

💡 ¿Por qué usamos pathlib?

pathlib.Path es la forma moderna de trabajar con rutas en Python. Es más legible que os.path y funciona igual en Windows, Mac y Linux. El método .iterdir() es más elegante que os.listdir().

Paso 2 — Clasificar con Claude (IA)

Aquí está el cerebro de nuestro agente. Enviamos la lista de archivos a Claude y él nos dice en qué categoría va cada uno.

1

Entender el Prompt

El prompt es la instrucción que le damos a Claude. Debe ser:

  • Claro: explicar exactamente qué queremos
  • Estructurado: pedir respuesta en formato JSON
  • Con ejemplos: mostrar el formato esperado
2

Agregar la función clasificar_con_claude()

Agrega esta función en organizador.py, después de escanear_carpeta():

def clasificar_con_claude(archivos: list[str]) -> dict:
    """
    Envía la lista de archivos a Claude para que los clasifique por categoría.
    
    Args:
        archivos: Lista de nombres de archivos.
    
    Returns:
        Diccionario con la estructura {nombre_archivo: categoria}.
    """
    client = anthropic.Anthropic(
        api_key=os.environ.get("ANTHROPIC_API_KEY")
    )
    
    # Construir la lista de archivos como texto
    lista_archivos = "\n".join(f"- {archivo}" for archivo in archivos)
    
    prompt = f"""Eres un asistente que clasifica archivos en categorías.

Analiza los siguientes nombres de archivos y clasifica CADA UNO en exactamente 
UNA de estas categorías:
- documentos (PDF, Word, texto, presentaciones)
- imagenes (fotos, capturas, diagramas)
- codigo (scripts, páginas web, estilos)
- audio_video (música, podcasts, videos, grabaciones)
- datos (hojas de cálculo, CSV, JSON, bases de datos)
- comprimidos (ZIP, RAR, TAR, 7Z)
- otros (archivos que no encajen en ninguna categoría anterior)

IMPORTANTE: Clasifica basándote en el NOMBRE y la EXTENSIÓN del archivo.
Si un archivo no tiene extensión, intenta deducir su tipo por el nombre.

Archivos a clasificar:
{lista_archivos}

Responde ÚNICAMENTE con un JSON válido con esta estructura exacta:
{{
    "nombre_archivo.ext": "categoria",
    "otro_archivo.ext": "categoria"
}}

No incluyas explicaciones, solo el JSON."""

    print("\n🤖 Enviando archivos a Claude para clasificación...")
    
    message = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=1024,
        messages=[
            {"role": "user", "content": prompt}
        ]
    )
    
    respuesta = message.content[0].text
    
    # Limpiar respuesta: a veces Claude envuelve el JSON en ```json ... ```
    respuesta_limpia = respuesta.strip()
    if respuesta_limpia.startswith("```"):
        # Remover bloques de código markdown
        lineas = respuesta_limpia.split("\n")
        lineas = [l for l in lineas if not l.strip().startswith("```")]
        respuesta_limpia = "\n".join(lineas)
    
    try:
        clasificacion = json.loads(respuesta_limpia)
        print(f"✅ Claude clasificó {len(clasificacion)} archivos:")
        for archivo, categoria in sorted(clasificacion.items()):
            print(f"   📄 {archivo} → 📁 {categoria}")
        return clasificacion
    except json.JSONDecodeError as e:
        print(f"❌ Error al parsear la respuesta de Claude: {e}")
        print(f"   Respuesta recibida: {respuesta}")
        return {}

🤖 Anatomía del Prompt

Analicemos las partes clave del prompt:

  • Rol: "Eres un asistente que clasifica archivos" — Le da contexto a Claude
  • Categorías definidas: Listamos las opciones válidas con ejemplos
  • Regla especial: "Si no tiene extensión, deduce por nombre" — Maneja edge cases
  • Formato de respuesta: Pedimos JSON exacto — Claude es muy bueno siguiendo formatos
  • Restricción: "No incluyas explicaciones" — Evita texto extra que rompa el parseo

Paso 3 — Mover Archivos

El paso final: actuar. Creamos las subcarpetas y movemos cada archivo a donde corresponde.

1

Agregar la función mover_archivos()

Agrega esta función en organizador.py, después de clasificar_con_claude():

def mover_archivos(ruta_carpeta: str, clasificacion: dict) -> dict:
    """
    Mueve los archivos a subcarpetas según la clasificación de Claude.
    
    Args:
        ruta_carpeta: Ruta a la carpeta que contiene los archivos.
        clasificacion: Diccionario {nombre_archivo: categoria}.
    
    Returns:
        Diccionario con estadísticas de archivos movidos por categoría.
    """
    carpeta = Path(ruta_carpeta)
    estadisticas = {}
    errores = []
    
    for archivo, categoria in clasificacion.items():
        origen = carpeta / archivo
        destino_carpeta = carpeta / categoria
        destino = destino_carpeta / archivo
        
        # Verificar que el archivo existe
        if not origen.exists():
            errores.append(f"No encontrado: {archivo}")
            continue
        
        # Crear la subcarpeta si no existe
        destino_carpeta.mkdir(exist_ok=True)
        
        # Mover el archivo
        try:
            shutil.move(str(origen), str(destino))
            estadisticas[categoria] = estadisticas.get(categoria, 0) + 1
        except Exception as e:
            errores.append(f"Error moviendo {archivo}: {e}")
    
    # Mostrar resumen
    print("\n📊 Resumen de organización:")
    print("-" * 40)
    for categoria, cantidad in sorted(estadisticas.items()):
        print(f"   📁 {categoria}: {cantidad} archivo(s)")
    print("-" * 40)
    total = sum(estadisticas.values())
    print(f"   ✅ Total movidos: {total}")
    
    if errores:
        print(f"\n⚠️ Errores ({len(errores)}):")
        for error in errores:
            print(f"   ❌ {error}")
    
    return estadisticas

💡 ¿Por qué usamos shutil.move()?

shutil.move() es más robusto que os.rename() porque puede mover archivos entre diferentes discos/particiones. Además, si el destino ya existe con el mismo nombre, genera un error en lugar de sobrescribir silenciosamente (lo cual es más seguro).

Paso 4 — Agente Completo

Unimos todo en una función main() con confirmación del usuario antes de mover archivos.

1

Reemplazar el bloque if __name__

def mostrar_plan(clasificacion: dict) -> None:
    """Muestra el plan de organización antes de ejecutar."""
    categorias = {}
    for archivo, categoria in clasificacion.items():
        if categoria not in categorias:
            categorias[categoria] = []
        categorias[categoria].append(archivo)
    
    print("\n" + "=" * 55)
    print("📋 PLAN DE ORGANIZACIÓN")
    print("=" * 55)
    for categoria, archivos in sorted(categorias.items()):
        print(f"\n📁 {categoria}/ ({len(archivos)} archivos)")
        for archivo in sorted(archivos):
            print(f"   └── {archivo}")
    print("\n" + "=" * 55)


def main():
    """Función principal del agente organizador."""
    print("=" * 55)
    print("🐍 AGENTE ORGANIZADOR DE ARCHIVOS CON IA")
    print("   Powered by Claude (Anthropic)")
    print("=" * 55)
    
    # Solicitar la ruta de la carpeta
    ruta = input("\n📂 Ingresa la ruta de la carpeta a organizar: ").strip()
    
    if not ruta:
        ruta = "desordenada"
        print(f"   Usando carpeta por defecto: '{ruta}'")
    
    # Paso 1: Escanear
    print("\n" + "-" * 55)
    print("PASO 1: Escaneando carpeta...")
    print("-" * 55)
    archivos = escanear_carpeta(ruta)
    
    if not archivos:
        print("\n🛑 No hay archivos para organizar. Finalizando.")
        return
    
    # Paso 2: Clasificar con Claude
    print("\n" + "-" * 55)
    print("PASO 2: Clasificando con Claude...")
    print("-" * 55)
    clasificacion = clasificar_con_claude(archivos)
    
    if not clasificacion:
        print("\n🛑 No se pudo clasificar. Finalizando.")
        return
    
    # Mostrar el plan
    mostrar_plan(clasificacion)
    
    # Pedir confirmación
    respuesta = input("\n¿Deseas ejecutar este plan? (s/n): ").strip().lower()
    
    if respuesta != "s":
        print("\n🚫 Operación cancelada por el usuario.")
        return
    
    # Paso 3: Mover archivos
    print("\n" + "-" * 55)
    print("PASO 3: Moviendo archivos...")
    print("-" * 55)
    estadisticas = mover_archivos(ruta, clasificacion)
    
    # Resultado final
    print("\n" + "=" * 55)
    print("🎉 ¡ORGANIZACIÓN COMPLETADA!")
    print("=" * 55)
    print(f"Revisa la carpeta '{ruta}' para ver el resultado.")


if __name__ == "__main__":
    main()

🤖 Modo Seguro: Confirmación antes de actuar

Nuestro agente muestra el plan completo y pide confirmación antes de mover cualquier archivo. Este es un patrón importante en agentes de IA: nunca ejecutar acciones destructivas sin aprobación humana. Es lo que se llama "Human in the Loop".

Re-crear los archivos de prueba

# Eliminar las subcarpetas creadas

Remove-Item -Recurse -Force desordenada

python crear_archivos_prueba.py

python organizador.py

✅ Los 3 escenarios deben funcionar sin errores

Si todos pasaron, tu agente maneja correctamente los casos normales y los edge cases.

Solución de Problemas

🔧 Errores Comunes

Error Causa Solución
AuthenticationError API Key inválida o no configurada Verificar la variable de entorno ANTHROPIC_API_KEY. Ejecutar echo $env:ANTHROPIC_API_KEY en PowerShell.
RateLimitError Demasiadas solicitudes a la API Esperar unos segundos y volver a intentar. Anthropic limita las solicitudes por minuto.
json.JSONDecodeError Claude no respondió con JSON válido Volver a ejecutar. Ocurre raramente. El código ya limpia bloques ```json```.
FileNotFoundError La carpeta no existe Verificar la ruta. Usar ruta relativa desde la raíz del proyecto.
PermissionError Sin permisos para mover archivos Ejecutar VS Code como Administrador o usar una carpeta donde tengas permisos.
ModuleNotFoundError: anthropic Librería no instalada o entorno virtual no activado Activar el entorno virtual (.\venv\Scripts\Activate.ps1) y ejecutar pip install anthropic.
shutil.Error: Destination path already exists Ya existe un archivo con el mismo nombre en la carpeta destino Eliminar las subcarpetas y volver a ejecutar, o usar archivos de prueba nuevos.

💡 Tips de Depuración

🔍 Verificar la API Key
  • PowerShell: echo $env:ANTHROPIC_API_KEY
  • Debe empezar con sk-ant-
  • Si está vacía, configúrala de nuevo
🐍 Verificar el Entorno Virtual
  • Debe aparecer (venv) en la terminal
  • pip list debe mostrar anthropic
  • Si no aparece, activar con .\venv\Scripts\Activate.ps1
📊 Ver la respuesta de Claude
  • Si el JSON falla, imprime respuesta antes del json.loads()
  • Verifica que Claude respondió con JSON puro
  • Ajusta el prompt si es necesario
💰 Controlar Costos
  • Cada ejecución usa ~500-800 tokens
  • Con claude-sonnet-4-20250514, el costo es mínimo (~$0.005 por ejecución)
  • Monitorea tu uso en console.anthropic.com

Reto Final — Extender el Agente

Ya tienes un agente funcional. Ahora, ¡mejóralo! Elige uno o más retos:

📝 Reto 1: Agregar Logging

Haz que el agente guarde un registro en log_organizacion.txt con:

  • Fecha y hora de ejecución
  • Archivos movidos y a qué carpeta
  • Errores encontrados

Pista: Usa el módulo datetime y escritura de archivos.

👀 Reto 2: Modo "Dry Run"

Agrega un parámetro --dry-run que:

  • Muestre qué haría el agente
  • NO mueva ningún archivo
  • Útil para verificar antes de actuar

Pista: Usa argparse y un flag booleano.

🔄 Reto 3: Soporte Recursivo

Modifica el agente para que también organice archivos dentro de subcarpetas:

  • Escanear recursivamente con rglob()
  • Mover archivos manteniendo la categorización
  • Evitar mover las carpetas de destino

Pista: Usa Path.rglob("*") y filtra .is_file().

↩️ Reto 4: Deshacer Organización

Crea una función que revierta la organización:

  • Guardar un "mapa" de movimientos en un JSON
  • Comando --undo para restaurar
  • Mover todo de vuelta a la carpeta raíz

Pista: Guarda {"origen": "destino"} al mover.
