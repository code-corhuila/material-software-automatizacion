✅ Probar tu Flujo

Es hora de validar que todo funciona correctamente.

Paso 1: Testeo Manual en n8n

1. Entra en modo Test

  • Haz clic en el botón "Test" (o "Execute node")
  • Selecciona el nodo Webhook
  • Ingresa datos de prueba en JSON:
    { "email": "tu@email.com", "nombre": "Juan", "mensaje": "Hola desde n8n!" }
  • Haz clic en "Send" o "Test"

Paso 2: Prueba con Postman

2. Usar Postman para enviar datos

  • Abre Postman (descárgalo desde postman.com)
  • Crea una nueva solicitud POST
  • URL: (la URL de tu Webhook de n8n)
  • Headers:
    Content-Type: application/json
  • Body (raw JSON):
    { "email": "tu@email.com", "nombre": "Cliente Prueba", "asunto": "Prueba de flujo n8n" }
  • Haz clic en "Send"

Paso 3: Verificar Ejecución

3. Revisar logs y resultados

  • En n8n, ve a la sección "Runs" o "Execution History"
  • Deberías ver:
    • El Webhook recibió datos ✓
    • El IF validó el email ✓
    • Gmail envió el correo ✓
  • Verifica tu bandeja de entrada esperando el correo
✅ ¡Éxito! Si recibes el correo, tu flujo está funcionando correctamente. Si no, verifica los logs y la sección de Troubleshooting.

Paso 4: Prueba con curl (Línea de Comandos)

Alternativa: Usar curl

Si prefieres la línea de comandos, ejecuta:

curl -X POST http://localhost:5678/webhook/tuwebhook \ -H "Content-Type: application/json" \ -d '{"email":"tu@email.com","nombre":"Prueba"}'
